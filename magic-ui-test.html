About Magic MCP

Magic MCP is a tool that helps create beautiful UI components using natural language descriptions. This page demonstrates how to use components generated by the Magic MCP server.

Note: The Magic MCP server must be properly configured and running for these examples to work.

Usage in Cline

To generate a UI component using Cline, use the following command format:

/ui create a [component description]

Examples:

  • /ui create a modern button with gradient background and hover effect
  • /ui create a testimonial card with avatar, quote, and user info
  • /ui create a responsive navigation bar with mobile menu

Example Component Integration

When Magic MCP generates a component, you can integrate it into your project like this:

<!-- 1. Include the component script -->
<script src="js/components/modern-button.js"></script>

<!-- 2. Use the custom element in your HTML -->
<modern-button variant="primary" size="medium">Click Me</modern-button>
<modern-button variant="secondary" size="small">Learn More</modern-button>
